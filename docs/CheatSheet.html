 Emacs Key Notations

C- control
M- alt
S- shift
DEL backspace
RET return
SPC spacebar
ESC escape
TAB tab

 Additional Tips

For example C-x s means that we must press and hold Control, then press x, release both keys, then press s.

The meta key typically means the alt key, but it can also be mapped to the Escape key, e.g. M-x means press and hold Alt and then press x. Alternatively, you can press and release Escape and then press x.

 Buffer List

We open the buffer list window by pressing C-x C-b and once there we can use the following keystrokes:

Keystroke Action Occurs
d Marks buffer for deletion When you press x
f Displayes selected buffer in current window Immediately
m Marks buffer to be displayed in another window When you press v
n Moves to the next buffer in the list (i.e., down one line). Immediately
p Moves to the previous buffer in the list (i.e., up one line). Immediately
q Quits buffer list Immediately
s Saves the selected buffer Immediately
u Unmarks the selected buffer Immediately
v Displayed marked buffers Immediately
x Execute commands on marked buffers Immediately
% Toggles read-only mode for the selected buffer Immediately
~ Marks the buffer as modified and do not save changes yet Immediately
1 Displays buffer in full screen Immediately
2 Displays buffer in next horizontal window Immediately

TAB Expands/Collabpses a section
c Commits stage changes
P Pushes committed changes upstream
h Shows a list of other magit popups
q Closes magit status buffer

 Additional Customizations

I have better-defaults and PDE installed; however they have conflicting keybindings configured for hippie-expand and dabbrev-expand. In order to fix this I had to disambiguate them in my ~/.emacs.d/init.el: 

            (global-set-key (kbd "M-/") 'hippie-expand)
              (global-set-key (kbd «M-;») 'dabbrev-expand)
